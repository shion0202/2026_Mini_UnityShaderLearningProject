# 2026_Mini_UnityShaderLearningProject
상용 셰이더 분석과 커스텀 구현을 통해 그래픽스 이해도와 셰이더 개발 역량을 높이 프로젝트

### 환경
- **유니티 버전:** `Unity 6.3`
- **렌더 파이프라인:** `URP`
- **구현 방식:** `Shader Graph`를 중심으로 하되, 필요에 따라 일부 기능을 `HLSL`로 구현
